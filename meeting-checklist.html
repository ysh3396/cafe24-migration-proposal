CAFE24 OVERVIEW

CAFE24에서 독립몰로, 왜?

현재: CAFE24 호스팅

CAFE24는 임대 아파트와 같습니다.
입주(가입)하면 바로 쇼핑몰을 운영할 수 있지만,
벽을 허물거나 구조를 바꿀 수는 없습니다.

✗ 도매 전용 기능(승인제, 등급별 가격) 구현 한계
✗ 사업자 A/B를 완전히 분리할 수 없음
✗ 데이터(회원/주문)를 직접 소유하지 못함
✗ 디자인/기능 커스터마이징에 한계
전환 후: 독립몰 (자가 소유)

독립몰은 내 건물을 짓는 것입니다.
원하는 구조로 자유롭게 설계하고,
모든 데이터를 직접 소유합니다.

✓ B2B 승인제, 등급별 가격, 세금계산서 자동발행
✓ 펠릭스/소독차 완전 분리 (별도 DB, 별도 관리자)
✓ 회원/주문/매출 데이터 100% 소유
✓ 원하는 기능 무제한 추가 가능

이관 과정에서 기존 회원 정보, 상품, 주문 내역, 리뷰는 모두 새 시스템으로 안전하게 옮겨집니다. 고객은 변화를 거의 느끼지 못합니다.

MIGRATION RISKS

이관 시 저희가 챙겨드릴 부분

CAFE24에서 독립몰로 전환할 때 발생할 수 있는 이슈를 미리 파악하고, 안전하게 처리합니다.

📦

데이터 이관

  • 회원/상품/주문/리뷰 데이터를 CAFE24에서 추출하여 새 시스템으로 옮깁니다
  • 기존 비밀번호는 보안상 가져올 수 없어 임시 비밀번호를 발급하고, 최초 로그인 시 변경을 안내합니다
  • 임시 PW를 알림톡/SMS로 발송하여 즉시 로그인 가능하게 처리합니다
🖼️

상품 이미지

  • 현재 상품 이미지는 CAFE24 서버에 저장되어 있습니다
  • CAFE24 해지 시 이미지가 사라지므로 해지 전에 전체 이미지를 안전하게 백업합니다
  • 새 서버로 이미지를 옮기고, 모든 링크를 자동 교체합니다
💳

결제(PG) 계약

  • CAFE24를 통해 결제를 사용 중이라면, 독립몰용으로 새 결제 계약이 필요합니다
  • 심사에 5~15 영업일이 걸리므로 계약 즉시 접수하여 일정을 맞춥니다
  • 이미 직접 계약이 있다면 그대로 사용 가능합니다
🌐

도메인 전환

  • 기존 도메인(felixliquids.com 등)을 새 서버로 연결합니다
  • 도메인에 연결된 이메일이 끊기지 않도록 사전 조치합니다
  • 전환 중에도 CAFE24를 병행 운영하여 안전하게 이관합니다
