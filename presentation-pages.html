SECTION 1 · 프로젝트 이해

클라이언트 현황 분석

🏬

CAFE24 현재 운영

도매 사이트 1개(felixliquids.com) 및 소매 사이트 2개(felixlab.co.kr, sodocha.com), 총 3개 쇼핑몰을 CAFE24 호스팅 기반으로 운영 중입니다.

⚠️

사업자 주체 혼재

사업자 A(펠릭스)와 사업자 B(소독차)가 동일 플랫폼에서 혼재 운영 중. 브랜드 독립성이 확보되지 않아 고객 혼란 및 관리 비효율이 발생합니다.

🔒

호스팅 기반 한계

CAFE24 호스팅 환경에서는 B2B 승인제 회원, 세금계산서 자동발행, 실시간 재고 연동 등 고급 커스텀 기능 구현에 제약이 있습니다.

🎯

독립몰 전환 기회

독립몰로 전환하면 완전한 기술 통제권을 확보하여 브랜드 분리, 고급 기능 구현, 데이터 독립 관리가 모두 가능해집니다.

SECTION 1 · 프로젝트 이해

프로젝트 목표

1

브랜드 독립성 확보

사업자 A(펠릭스)와 사업자 B(소독차)를 완전히 분리하여 각각 독립된 브랜드 아이덴티티를 구축합니다.

2

이용 경험 분리

도매(B2B) 고객과 소매(B2C) 고객의 완벽한 이용 경험 분리로 각 고객군에 최적화된 서비스를 제공합니다.

3

관리 효율 증대

사업자별 독립된 관리자 페이지(총 2벌)로 운영 효율을 극대화하고, 권한 분리로 보안을 강화합니다.

4

보안 강화

도매 고객 정보 및 가격 정보를 완벽하게 보호하여 B2B 거래의 기밀성을 확보합니다.

SECTION 3 · 핵심 기능

B2B 도매몰 핵심 기능

🔐

승인제 회원가입

사업자등록증 업로드 → 관리자 승인 후 이용 가능. 미승인 회원은 상품 열람만 가능하며 가격 및 주문 기능에 접근할 수 없습니다.

👁️‍🗨️

가격 비노출 설정

로그인한 승인 회원에게만 도매가를 노출합니다. 비로그인/미승인 상태에서는 "로그인 후 확인" 메시지를 표시합니다.

🧾

세금계산서 자동발행

홈택스 API 또는 바로빌 API와 연동하여 주문 완료 시 세금계산서를 자동 발행합니다. 수동 발행 없이 정산을 자동화합니다.

🧮

현금영수증 자동발행

현금 결제 시 현금영수증 자동발행 API를 연동하여 세무 처리를 간소화합니다. 관리자 페이지에서 발행 내역을 확인합니다.

SECTION 3 · 핵심 기능

공통 쇼핑몰 기능

💳

결제 시스템

토스페이먼츠/NHN KCP PG 연동, 무통장 입금 및 자동 입금 확인 기능을 지원합니다.

📦

주문/배송 관리

송장 번호 대량 업로드, CJ대한통운/한진택배 API 연동으로 배송 추적을 자동화합니다.

📊

재고 관리

실시간 재고 차감 로직, 품절 시 카카오 알림톡/SMS 자동 발송으로 운영 효율을 높입니다.

👥

회원 관리

회원 등급제, 적립금, 쿠폰 시스템을 지원하여 고객 로열티를 관리합니다.

SECTION 3 · 핵심 기능

CAFE24 데이터 이관 전략

1

DB 추출

CAFE24 API/Export로 회원, 상품, 주문, 리뷰 데이터를 추출합니다.

2

매핑 설계

기존 CAFE24 스키마 → 신규 PostgreSQL 스키마로의 매핑 테이블을 설계합니다.

3

무결성 검증

중복 데이터 체크, 데이터 타입 검증, 누락 항목 확인으로 이관 품질을 보장합니다.

4

이관 실행

기존 회원 ID를 유지하되 비밀번호는 신규 생성 기반으로 재설정. 이관 완료 안내 메일을 발송합니다.

SECTION 3 · 핵심 기능

관리자 대시보드

📈

매출 대시보드

일간/월간 매출, 신규 가입자, 미입력 송장 현황 등을 시각적 차트로 한눈에 파악합니다.

🏆

통계 시스템

상품별 판매 순위, 등급별 매출 비중 분석으로 데이터 기반 의사결정을 지원합니다.

🔑

권한 관리

운영진별 접근 가능 메뉴(상품 관리, 정산, 회원 등)를 분리하여 보안을 강화합니다.

📥

데이터 다운로드

가입자 내역, 실시간 재고, 매출 리포트 등을 엑셀로 추출하여 오프라인 분석에 활용합니다.
